# AI Code Assistant

An AI-powered tool that explains code, generates comments, and finds bugs using LLM.

## Features

- **Explain Code**: Get clear explanations of what code does
- **Generate Comments**: Automatically add inline comments
- **Bug Detection**: Find bugs and security issues

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Add your OpenAI API key to `.env`

## Usage

```bash
python code_assistant.py
```

## Skills Demonstrated

- LLM prompting
- Function calling / tool calling
- OpenAI API integration
